					Структура
	Каждый из проектов размещен в отдельной папке в соотвествии с заданием:
		1. Уравнение Фредгольма
		2. Уравнение Вольтерра
		3. Решение интегральных уравнений методом Ньютона
		4. Решение нелинейного интегрального уравнения из системы уравнений методом Ньютона
		5. Решение интегро-дифференциального уравнения
		6. TODO

==============================================================================================
					How to:
	В начале каждого кода заданы параметры
		1. Функция ядра K
		2. Граничные значения
		3. Функции ...
		4. Константы
	Значения которых взяты в репрезитории для варианта 6.

	Скачивается zip файл копии репрезитория и распаковывается в необходимую директорию
	Файлы в репрезиториях хранятся в виде кода ядра, при скачивании zip архива, файлы 
	будут иметь обычный синтаксис Wolfram
